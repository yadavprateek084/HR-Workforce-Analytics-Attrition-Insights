# 📌 Project Overview

This project focuses on building an interactive **HR Analytics Dashboard** using **Power BI** to analyze employee attendance trends, work-from-home patterns, leave utilization, and workforce presence insights.

The dashboard transforms raw attendance records into meaningful business intelligence that helps HR teams and management:

* Monitor employee attendance behavior
* Track remote work trends
* Analyze leave consumption patterns
* Improve workforce planning and operational efficiency
* Support data-driven HR decisions

---

# 🧠 Business Problem

HR departments often struggle with fragmented attendance records stored across spreadsheets, making it difficult to:

* Identify absenteeism trends
* Track workforce availability
* Measure employee presence rates
* Analyze remote work adoption
* Generate quick HR reports for leadership

This project solves that problem by creating a centralized and visually interactive analytics solution in Power BI.
